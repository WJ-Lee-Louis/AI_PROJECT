# HICO-DET sample-small images

Source: https://huggingface.co/datasets/AgressivniyKot/hico-det-sample-small

Purpose in this project:
- Quick image-only sanity checks for human-object interaction behavior prompts.
- This is a small Hugging Face sample mirror, not the full HICO-DET dataset.

Only images were extracted from the Parquet files; annotations are not required for the current reference-image inference checks.
