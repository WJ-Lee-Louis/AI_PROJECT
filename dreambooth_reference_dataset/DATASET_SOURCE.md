# DreamBooth reference dataset

Source: https://github.com/google/dreambooth
Copied folder: `dataset/`

This folder is intended for quick reference-image personalization checks.
The original repository describes the dataset as 30 subjects across 15 classes with 4-6 images per subject.

License/source metadata from the original dataset is preserved in:

- `references_and_licenses.txt`
- `prompts_and_classes.txt`
